# Cloud Computing Performance Metrics Abalysis
 
 This is a repo dedicated to exploratory data analysis of a dataset about CC performance metrics. 

 
 # How to set up

 ### Linux
 - Clone the repo.
 - Give execution permission to both .sh files.S
 - Run "venv_setup.sh"
